    Game Functions¶

    Core game function reference.

    Maze Generation¶

    generateMaze()¶

    Generate new maze and place player/goal.

    generateMaze();

    Game Loop¶

    resetGame()¶

    Reset game state for new maze.

    resetGame();

    checkVictory()¶

    Check if player reached goal.

    checkVictory();

    Data Management¶

    saveRun(time)¶

    Save completed run to leaderboard.

    saveRun(45230); // time in milliseconds

    addXP(amount)¶

    Award XP and update level.

    addXP(50); // amount in XP points

    Utilities¶

    formatTime(ms)¶

    Format milliseconds to MM:SS.MS.

    formatTime(45230); // Returns "00:45.230"

    sanitizeUsername(name)¶

    Clean and validate username.

    sanitizeUsername("Player123"); // Returns sanitized name

    Rendering¶

    renderMaze()¶

    Draw maze to screen.

    renderMaze();

    playReplay(run)¶

    Play back saved run.

    playReplay(leaderboard[0]);

    All functions are internal to index.html.
