    Architecture Overview¶

    SuperMaze is built with a clean separation between frontend, server, and data layers.

    System Overview¶

    • Frontend: Vanilla HTML5/CSS3/JavaScript
    • Server: Node.js + Express + WebSocket
    • Storage: localStorage (client) + JSON files (server)
    • Styling: Bootstrap 5 + custom CSS

    Component Layers¶

    Presentation Layer¶

    • HTML5 UI
    • CSS3 styling
    • Responsive design

    Game Logic Layer¶

    • Maze generation
    • Player movement
    • Collision detection
    • Scoring system

    Data Layer¶

    • localStorage persistence
    • JSON serialization
    • Server APIs

    Communication Layer¶

    • WebSocket for multiplayer
    • HTTP for leaderboard
    • Token-based auth

    See Core Components for detailed information.
