    Technology Stack¶

    The technologies and frameworks used in SuperMaze.

    Frontend Stack¶

    Technology Version Purpose
    HTML5 Latest Markup and structure
    CSS3 Latest Styling and layout
    JavaScript ES6+ Game logic and interactivity
    Bootstrap 5 5.3.2 UI framework

    Backend Stack¶

    Technology Version Purpose
    Node.js 18+ Runtime environment
    Express 5.1.0+ Web server framework
    WebSocket (ws) 8.18.3+ Real-time communication

    Databases and Storage¶

    Technology Purpose
    localStorage Client-side game data
    JSON files Server-side leaderboard
    Session memory Active player tracking

    Development Tools¶

    Tool Purpose
    npm Package manager
    Git Version control
    TypeScript (optional) Type safety
    ESLint (optional) Code quality

    Build and Deployment¶

    Tool Purpose
    MkDocs Documentation generation
    Material theme Documentation styling
    npm scripts Build automation

    Browser Compatibility¶

    Supported Browsers¶

    • Chrome 90+
    • Edge 90+
    • Firefox 88+
    • Safari 14+

    Required APIs¶

    • localStorage
    • WebSocket
    • fetch (optional, for multiplayer)
    • Canvas (for rendering)

    No External Dependencies¶

    SuperMaze frontend has zero dependencies:

    • No jQuery
    • No React
    • No Vue
    • No Svelte
    • No external libraries

    Pure vanilla JavaScript and CSS.

    Optional Tools¶

    For development:

    • TypeScript compiler
    • Prettier (formatting)
    • ESLint (linting)
    • Jest (testing)

    Simple, lightweight, and powerful technology choices.
