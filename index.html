    SuperMaze Documentation¶

    Welcome to the official SuperMaze documentation. This guide covers everything you need to know about the SuperMaze game engine, from getting started to advanced development.

    What is SuperMaze?¶

    SuperMaze is a procedurally-generated maze game built with vanilla HTML5, CSS3, and JavaScript. It features both singleplayer and multiplayer modes, with real-time player tracking, customizable themes, and a comprehensive settings system.

    Quick Navigation¶

    • New to SuperMaze? Start with Getting Started
    • Want to play? Check out Quick Start
    • Need API details? See API Reference
    • Building a custom version? Read Development

    Key Features¶

    • Dynamic maze generation with adjustable difficulty
    • Customizable theme system with personal backgrounds
    • Game settings editor with JSON support
    • Real-time multiplayer via WebSocket
    • Leaderboard with replay functionality
    • Experience points and level tracking
    • Secure input validation and authentication

    System Requirements¶

    • Modern web browser (Chrome, Edge, Firefox, Safari)
    • Node.js 18+ (for multiplayer server)
    • No external dependencies required

    FAQ¶

    Can I play offline?¶

    Yes! The singleplayer mode works completely offline. Your scores and progress are saved locally in your browser.

    How do I host a multiplayer server?¶

    Install Node.js, navigate to src/server, and run: 

    npm install
npm start

    The server will run on http://localhost:3000

    Can I customize the maze difficulty?¶

    Yes! Use the Settings button in the game to adjust difficulty (Easy, Medium, Hard) and graphics quality (Low, Mid, High, Advanced).

    How are my scores saved?¶

    All your progress is saved locally in your browser's localStorage. No data is sent to external servers unless you choose to enable multiplayer.

    Can I create custom themes?¶

    Absolutely! Use the theme customization button to create themes with custom colors and background images.

    Is SuperMaze open source?¶

    Yes! Visit the GitHub repository for the full source code.
