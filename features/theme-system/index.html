    Theme System¶

    Personalize SuperMaze with themes and custom styling.

    Built-in Themes¶

    Light¶

    • Bright gray gradient background
    • Dark text for readability
    • Professional appearance

    Dark¶

    • Dark gray/black background
    • Light text
    • Easy on the eyes

    Blue¶

    • Navy to cyan gradient
    • Cool color scheme
    • Calm atmosphere

    Green¶

    • Forest green gradient
    • Natural appearance
    • Soothing tones

    Pink¶

    • Magenta gradient
    • Vibrant and energetic
    • Modern aesthetic

    Red¶

    • Dark red gradient
    • Warm tones
    • Bold appearance

    Selecting a Theme¶

    1. Click the theme dropdown on the home screen
    2. Select your preferred theme
    3. Theme changes immediately
    4. Selection is saved automatically

    Custom Themes¶

    Create unlimited custom themes with full control over colors and backgrounds.

    Custom Theme Properties¶

    {
  "bg1": "#001f3f",
  "bg2": "#003d99",
  "cardBg": "rgba(0, 31, 63, 0.9)",
  "text": "#e0f2ff",
  "title": "#00d4ff",
  "border": "rgba(0, 150, 255, 0.3)",
  "bgImage": ""
}
    Property Type Description
    bg1 Hex Color Gradient start color
    bg2 Hex Color Gradient end color
    cardBg Color Card/dialog background
    text Hex Color Main text color
    title Hex Color Title/header color
    border Color Border color
    bgImage URL Background image (optional)

    Creating a Custom Theme¶

    Option 1: Use Customizer Button¶

    1. Click the customize button (plus icon) next to theme dropdown
    2. Adjust colors and image URL
    3. Preview changes in modal
    4. Click "Save Theme"
    5. New theme appears in dropdown

    Option 2: Edit JSON Directly¶

    1. Open Settings
    2. Find maze_custom_theme in localStorage
    3. Edit colors and properties
    4. Save and refresh browser

    Color Formats¶

    Hex Colors¶

    #FF0000  (red)
#00FF00  (green)
#0000FF  (blue)
#FFFFFF  (white)
#000000  (black)

    RGBA Colors¶

    rgba(255, 0, 0, 0.5)      (semi-transparent red)
rgba(0, 0, 0, 0.3)        (semi-transparent black)
rgba(255, 255, 255, 0.1)  (almost transparent white)

    Background Images¶

    Adding an Image¶

    1. Host image online (imgur, imgbb, etc.)
    2. Get direct image URL
    3. Paste URL in customizer
    4. Leave blank to use gradient instead

    Recommended Specifications¶

    • Format: JPEG or PNG
    • Dimensions: 1920x1080 or larger
    • File Size: 1-3 MB
    • Aspect Ratio: 16:9

    Image Hosting Services¶

    • Imgur - Free hosting
    • ImgBB - Permanent storage
    • Cloudinary - Professional CDN
    • Pixabay - Free images

    Theme Persistence¶

    Themes are saved to localStorage:

    • Key: maze_theme (selected theme)
    • Key: maze_custom_theme (custom theme data)
    • Loaded automatically on startup
    • Survives page refreshes

    Advanced Customization¶

    Using Transparency¶

    Create layered effects with transparency:

    {
  "cardBg": "rgba(0, 0, 0, 0.5)",  // 50% transparent black
  "text": "rgba(255, 255, 255, 0.9)" // 90% opaque white
}

    Gradient Backgrounds¶

    Both bg1 and bg2 are blended as a 135-degree gradient:

    bg1 ↘ bg2
↙ ↗
135° angle

    Accessibility¶

    Ensure sufficient contrast:

    • Bright text on dark background
    • Dark text on light background
    • Avoid red/green combinations for colorblind users
