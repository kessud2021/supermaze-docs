    Leaderboard¶

    Track and compete with your best maze times.

    Accessing the Leaderboard¶

    Click the "Leaderboard" button on the home screen to view all your completed runs.

    Leaderboard Features¶

    View All Runs¶

    See every maze you've completed with: - Username - Completion time - Date played - Sorted by fastest time

    Replay Any Run¶

    Click on any leaderboard entry to: - Watch your exact moves - See your completion time - Review your performance - Learn from past attempts

    Run Information¶

    Each leaderboard entry contains:

    Field Description
    Username Player name from login
    Time MM:SS.MS format
    Date When completed
    Moves Complete move history

    Time Format¶

    Times display in MM:SS.MS format:

    • MM = Minutes (00-59)
    • SS = Seconds (00-59)
    • MS = Milliseconds (000-999)

    Examples¶

    • 00:45.230 = 45 seconds, 230 milliseconds
    • 02:15.600 = 2 minutes, 15 seconds, 600 milliseconds
    • 01:00.000 = 1 minute exactly

    Sorting¶

    Leaderboard automatically sorts by:

    1. Fastest Time First
    2. Slowest Time Last

    Trying to beat your personal best!

    Replaying Runs¶

    How to Replay¶

    1. Open Leaderboard
    2. Click on any run
    3. Maze screen loads with your moves
    4. Watch your recorded moves play back

    Replay Speed¶

    • Fixed playback speed (100ms per move)
    • Consistent for all replays
    • Shows complete move sequence
    • Time counter displays elapsed time

    Learning from Replays¶

    Use replays to: - Identify faster routes - Understand maze layout - Find shortcuts - Improve technique

    Local Storage¶

    Leaderboard Storage¶

    Leaderboard is stored in localStorage:

    • Key: maze_leaderboard
    • Format: JSON array
    • Auto-saved after each run
    • Survives page refreshes

    Example Storage Format¶

    [
  {
    "username": "Player1",
    "time": 45230,
    "moves": [...],
    "xp": 50,
    "date": "2026-01-16T10:30:00Z",
    "mazeSnapshot": [...],
    "startPos": {"x": 1, "y": 1}
  }
]

    Managing Your Leaderboard¶

    Clearing Leaderboard¶

    To reset your leaderboard:

    1. Open browser developer tools (F12)
    2. Go to Console
    3. Run: localStorage.removeItem('maze_leaderboard')
    4. Refresh browser

    Exporting Runs¶

    Copy your leaderboard data:

    1. Open Console (F12)
    2. Run: console.log(JSON.stringify(JSON.parse(localStorage.getItem('maze_leaderboard')), null, 2))
    3. Copy the output
    4. Save to a text file

    Importing Runs¶

    To restore from backup:

    1. Open Console (F12)
    2. Run: localStorage.setItem('maze_leaderboard', '[...paste your JSON...]')
    3. Refresh browser
